# Connections

A clone of the New York Times Connections game.

## How to Play

- Click on words to select them (up to 4).
- Click Submit to check if they form a category.
- If correct, the category is revealed and those words are removed.
- If incorrect, it's a mistake (max 4).
- Goal: Find all 4 categories.

## Configuration

Edit `config.json` to change the categories and words. Each category should have a name and exactly 4 words.

## Running

Open `index.html` in a web browser. For local development, serve the files with a local server (e.g., `python -m http.server` or `npx http-server`).
